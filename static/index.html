Acerca de las Series de McLaurin y Taylor

Las series de Taylor son representaciones de funciones como sumas infinitas de términos calculados a partir de los valores de las derivadas de la función en un punto. Las series de McLaurin son un caso especial de las series de Taylor centradas en el punto x = 0.

Serie de Taylor

La serie de Taylor de una función f(x) alrededor de un punto x = a está dada por:

f(x) = f(a) + f'(a)(x-a) + f''(a)(x-a)²/2! + f'''(a)(x-a)³/3! + ...

Serie de McLaurin

La serie de McLaurin es simplemente la serie de Taylor centrada en a = 0:

f(x) = f(0) + f'(0)x + f''(0)x²/2! + f'''(0)x³/3! + ...

Ayuda

Cómo usar esta aplicación

Para series de McLaurin:
  1. Seleccione una función de la lista desplegable.
  2. Ingrese el valor de x que desea evaluar.
  3. Especifique el número de términos para la aproximación.
  4. Haga clic en "Calcular" para ver los resultados.
Para series de Taylor:
  1. Seleccione una función de la lista desplegable.
  2. Ingrese el punto de convergencia (a).
  3. Ingrese el valor de x que desea evaluar.
  4. Especifique el número de términos para la aproximación.
  5. Haga clic en "Calcular" para ver los resultados.
Interpretación de resultados:
  • Valor exacto: El valor real de la función en el punto x.
  • Aproximación: El valor calculado usando la serie.
  • Error absoluto: La diferencia entre el valor exacto y la aproximación.
  • Error relativo: El error expresado como porcentaje del valor exacto.
